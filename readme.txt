# Project Name

¡Descúbreme! **Guess Who like game**

## About

This project is a web-based game inspired by the classic board game "Guess Who." 
It was created as a personal project to enhance my JavaScript programming skills and gain experience in web development.

## Features

- **Winning Celebration:** When you win the game, a festive confetti effect will appear on the screen to celebrate your victory.

## Usage

To use the application effectively, follow the instructions that appear when you click the information (ℹ️).
